# David Raya Premier League Goalkeeping Analysis (2024/25 vs 2025/26)

A football analytics dashboard built in Power BI comparing David Raya's Premier League performances across the 2024/25 and 2025/26 seasons. This project focuses on goalkeeping performance, shot-stopping efficiency, distribution, defensive actions, and season-over-season trends using data visualization and statistical analysis.

## Project Objectives

* Compare David Raya's performance across two Premier League seasons.
* Analyze key goalkeeping metrics such as saves, save percentage, goals conceded, and clean sheets.
* Evaluate shot-stopping effectiveness using Goals Prevented.
* Assess distribution quality through passing and long-ball metrics.
* Examine sweeper-keeper actions and aerial command.
* Present insights through an interactive Power BI dashboard.

## Dashboard Features

* KPI Cards for key performance metrics
* Goalkeeping Performance Comparison
* Shot-Stopping Efficiency Analysis
* Distribution Analysis
* Passing Accuracy Trend (Line Chart)
* Sweeper Keeper Action Distribution
* Radar Chart Season Comparison
* Errors & Discipline Analysis
* Key Insights and Performance Summary

## Key Findings

* Clean Sheets increased from 13 to 19 (+46.2%).
* Goals Conceded decreased from 34 to 26 (-23.5%).
* Save Percentage declined from 71.7% to 69.8%.
* Goals Prevented dropped from +1.99 to -1.13.
* Long Ball Accuracy improved from 32.9% to 34.2%.
* Sweeper Actions remained consistent across both seasons.
